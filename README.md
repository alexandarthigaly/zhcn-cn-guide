# zhcn-cn-guide

## 项目简介

zhcn-cn-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面不针对任何特定域名或网站，而是以通用方式收录了各类静态内容，便于浏览、参考和分发。

本仓库旨在提供一个结构化的页面集合，供有需要的用户自行查阅。所有内容均以 HTML 格式存储，可直接在浏览器中打开。

## 目录说明

```
/
├── index.html          # 入口页面，提供导航
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 资源文件（如样式、图片等）
    ├── style.css
    └── ...
```

- `index.html`：仓库的首页，可作为浏览页面的起点。
- `pages/`：包含所有独立 HTML 页面，文件名具有描述性。
- `assets/`：存放页面所需的静态资源，如 CSS、JavaScript 或图像文件。

## 页面归档说明

本仓库中的页面均为独立 HTML 文件，无需服务器环境即可在本地直接打开。每个页面内容经过整理，保持结构清晰，便于快速阅读或二次使用。

页面内容不涉及特定域名或网站，仅作为通用信息归档。用户可根据文件名和导航自行选择查看。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `index.html` 或 `pages/` 目录下的任一 HTML 文件。
3. 在浏览器中直接查看页面内容。
